## 概念

采用 flex 布局的元素为 flex container，其中的所有子元素为 flex item。

对于 div 类元素设置 `display` 属性为`flex`（或者是对 span 类元素设置 `display` 属性为 `inline-flex`），就可以将其设置为 flex container，其中子元素也就自动变成 flex item 了。

flex 布局时，可以指明是横向(row)布局，还是竖向(column)布局。两种不同的布局方式，会决定其他 flex 属性的效果。

![](https://drafts.csswg.org/css-flexbox-1/images/flex-direction-terms.svg)

*main axis*: 是 flex container 的主轴，其方向与 flex 布局方向一致。就是说，如果采用横向布局，则 main axis 与 CSS 坐标系的 *x* 轴一致，如果采用竖向布局，则 *main axis* 与 *y* 轴一致。

*corss axis*: 是 flex container 的交叉轴，与主轴垂直。

*main-start* / *main-end*: 分别是 flex container 主轴的起始边和终点边。默认情况下，flex item 从 *main-start* 向 *main-end* 方向排列。

*cross-start* / *cross-end*: 分别是 flex container **交叉轴**的起始边和终点边。默认情况下，flex item 在交叉轴方向上会占满 *cross-start* 和 *cross-end* 之间的空间。

*main size*: 指 flex **item** 的 **宽度** 或 **高度**，如果是横向布局，则 *main size* 是指 **宽度**，如果是竖向布局，则 *main size* 是指 **高度**。

*cross size*: 与 *main size* 类似。如果是横向布局，则 *cross size* 指 **高度**，如果是竖向布局，则 *cross size* 是指 **宽度**。

/ **

container 中有两根轴，一根叫主轴(main axis)，另外一根叫交叉轴(cross axis)。如果 container 采用横向布局，则水平方向为 main axis，垂直方向为 cross axis；如果 container 采用竖向布局，则垂直方向为 main axis，水平方向为 cross axis。

flex container 有以下几种可以使用的 flex 属性

  * flex-direction
  * flex-wrap
  * flex-flow
  * justify-content
  * align-items
  * align-content

## Container

### `flex-direction`

`flex-direction` 决定布局方向，也就是 main axis 的方向。

可使用以下几个值：

```css
.container {
  flex-direction: row | row-reverse | column | column-reverse;
}
```

`row` 为横向布局，该值为默认值

{% endblock %}

{% block flex-direction-row %}
.container {
  flex-direction: row;
}
{% endblock %}

{% block markdown %}

`row-reverse` 为横向布局，但其中的 flex item 会反方向铺展开，起点也是从最右侧开始。

{% endblock %}

{% block flex-direction-row-reverse %}
.container {
  flex-direction: row-reverse;
}
{% endblock %}

{% block markdown %}

`column` 为竖向布局。

{% endblock %}

{% block flex-direction-column %}
.container {
  flex-direction: column;
}
{% endblock %}

{% block markdown %}

`column-reverse` 竖向布局。与 `row-reverse` 类似，其中的 flex item 也会发反方向铺展开来，起点在下测。

{% endblock %}


{% block flex-direction-column-reverse %}
.container {
  flex-direction: column-reverse;
}
{% endblock %}

{% block markdown %}

### `flex-wrap`

当 container 的 main axis 上排不下所有的 flex item 时怎么办呢？ 属性 `flex-wrap` 就是来解决这个问题的，这个属性可以用来决定排不下的 item 是不是要换行或者换列。其可用值如下：

```css
.container {
  flex-wrap: nowrap | wrap | wrap-reverse;
}
```

`nowrap` 为默认值，表示不换行。(如果是采用竖向布局，则表示不换列 :)

如果 `item` 在主轴上排不下，比如下面的例子中，container 使用了 `column` 方向排序，内部的所有 item 的 height 加在一起已经超过了 containter 的可用高度了。但是因为 `flex-wrap` 的值为 `nowrap` 所以并不会换列。

{% endblock %}
{% block flex-wrap-nowrap %}
.container {
  flex-direction: column;
  flex-wrap: nowrap;
}

.item {
  min-height: 25px;
  width: 30px;
}
{% endblock %}


{% block markdown %}

`flex-wrap`的值 是 `wrap` 时， 如果所有 item 不能在主轴上一行或者一列排开，则会换行或者换列。

下面的例子中，采用竖向布局，主轴在垂直方向。因为 item 的 `min-height` 设为 `40px`，所以这些 item 并不能在一列排下。

{% endblock %}
{% block flex-wrap-wrap %}
.container {
  flex-direction: column;
  flex-wrap: wrap;
}

.item {
  min-height: 40px;
  width: 30px;
}
{% endblock %}

{% block markdown %}

`wrap-reverse`

{% endblock %}
{% block flex-wrap-wrap-reverse %}
.container {
  flex-direction: column;
  flex-wrap: wrap-reverse;
}

.item {
  min-height: 40px;
  width: 30px;
}
{% endblock %}



{% block markdown %}

### `flex-flow`

`flex-flow` 属性是 `flex-direction` 属性和 `flex-wrap` 属性的简写形式，默认值为 `row nowrap`

```css
.container {
  flex-flow: <flex-direction> || <flex-wrap>;
}
```

### `justify-content`

`justify-content` 用于描述 flex item 及其之间的"空格"在 container 的**主轴**上的对齐方式。

```css
.container {
  justify-content: flex-start | flex-end | center | space-between | space-around;
}
```

`flex-start` 是默认值，表示 container 中的内容会分布在主轴的起始位置。

{% endblock %}


{% block justify-content-flex-start %}
.container {
  justify-content: flex-start;
}

{% endblock %}

{% block markdown %}

`flex-end` 则表示 container 中的内容会分布在主轴的末尾位置。

{% endblock %}
{% block justify-content-flex-end %}
.container {
  justify-content: flex-end;
}

{% endblock %}

{% block markdown %}

`center` 则表示 container 中的内容在主轴上居中。

{% endblock %}
{% block justify-content-center %}
.container {
  justify-content: center;
}

{% endblock %}

{% block markdown %}

`space-between` 则表示 container 中主轴上的所有“空格”会均匀分布到 item 之间。

{% endblock %}
{% block justify-content-space-between %}
.container {
  justify-content: space-between;
}

{% endblock %}

{% block markdown %}

`space-around` 则表示 container 中主轴上的所有“空格”会均匀分布到 item 的**两侧**。

{% endblock %}
{% block justify-content-space-around %}
.container {
  justify-content: space-around;
}

{% endblock %}

## `align-items`

`align-items` 属性决定 container 中的 item 在 **交叉轴cross axis** 上是如何分布的。

可用值：

```css
.container {
  align-items: flex-start | flex-end | center | baseline | stretch;
}
```

`flex-start`表示对其到 交叉轴croass axis 的起始边(cross-start margin)。

{% endblock %}
{% block align-items-flex-start %}
.container {
  align-items: flex-start;
}
{% endblock %}


{% block markdown %}
`flex-end`表示对其到 交叉轴croass axis 的终点边(cross-end margin)。

{% endblock %}
{% block align-items-flex-end %}
.container {
  align-items: flex-end;
}
{% endblock %}


{% block markdown %}
`center`表示对其到 交叉轴croass axis 的中线。

{% endblock %}
{% block align-items-center %}
.container {
  align-items: center;
}
{% endblock %}



{% block markdown %}
`baseline`表示按照 item 的基线对齐。

{% endblock %}
{% block align-items-baseline %}
.container {
  align-items: baseline;
}

.item {
  font-size: 20px;
}

.item:nth-child(2n+1) {
  font-size: 28px;
}

.item:nth-child(4) {
  font-size: 12px;
}
{% endblock %}


{% block markdown %}
`stretch` 是默认值，如果 item 在的 height 没指明的话， item 会伸展开来，cross axis 方向填满。

{% endblock %}
{% block align-items-stretch %}
.container {
  align-items: stretch;
}

.item {
  height: auto;
}
{% endblock %}



{% block markdown %}

### `align-content`

`align-content` 这个属性只有在 item 排成多行(`flex-direction` 是横向)或者多列(`flex-direction` 是竖向)的时候才生效。它决定这多行/多列的 item 整体上在 cross axis 上的分布方式。

可用值：

```css
.container {
  align-content: flex-start | flex-end | center | space-between | space-around | stretch;
}
```

`flex-start` 表示所有 item 整体上分布在 cross axis 的起始位置

{% endblock %}
{% block align-content-flex-start %}
.container {
  flex-direction: column;
  flex-wrap: wrap;
  align-content: flex-start;
}
{% endblock %}




{% block markdown %}

`flex-end` 表示所有 item 整体上分布在 cross axis 的末尾位置

{% endblock %}
{% block align-content-flex-end %}
.container {
  flex-direction: column;
  flex-wrap: wrap;
  align-content: flex-end;
}
{% endblock %}



{% block markdown %}

`center` 表示所有 item 整体上在 cross axis 的中线上对齐

{% endblock %}
{% block align-content-center %}
.container {
  flex-direction: column;
  flex-wrap: wrap;
  align-content: center;
}
{% endblock %}



{% block markdown %}

`space-between` 表示所有每行/每列之间空间均匀分布。

{% endblock %}
{% block align-content-space-between %}
.container {
  flex-direction: column;
  flex-wrap: wrap;
  align-content: space-between;
}
{% endblock %}



{% block markdown %}

`space-around` 表示所有每行/每列两边的空间均匀分布。

{% endblock %}
{% block align-content-space-around %}
.container {
  flex-direction: column;
  flex-wrap: wrap;
  align-content: space-around;
}
{% endblock %}


{% block markdown %}

`stretch` 是默认值， 表示**每**行/列都将会自由拉伸开，cross axis 上占用整个空间。

当然拉伸的前提是，这个 item 在这个维度上的 size 没有作限制，比如下面例子中，第 3 个 item 限制了  `width` 则它不会 stretch。

{% endblock %}
{% block align-content-stretch %}
.container {
  flex-direction: column;
  align-content: stretch;
}

.item {
  width: auto;
}

.item:nth-child(3) {
  width: 50px;
}
{% endblock %}

## item

### `order`

默认情况下，flex item 在 container 中分布时会按照 html 代码中的顺序。但我们也可以使用 `order` 属性来控制 item 的分布顺序。


`order` 的值是一个整数。初始默认值是`0`,值越小，就会排的越靠前，这个值也可以是负值。

```css
.item {
  order: <integer>;
}
```

注意观察下面例子中 item 的顺序。

{% endblock %}
{% block order %}
.item:nth-child(2n) {
  order: -1;
}

.item:nth-child(5) {
  order: 3;
}

.item:nth-child(3) {
  order: 99;
}
{% endblock %}

### `flex-grow`

`flex-grow` 属性决定这个 item 所占空间比例

其值是一个不小于0的数字，注意该值并不是作为百分比来解释。默认值是0。

```css
.item {
  flex-grow: <number>; /* default 0 */
}
```

{% endblock %}
{% block flex-grow %}
.item:nth-child(1) {
  flex-grow: 1;
}

.item:nth-child(3) {
  flex-grow: 2;
}

.item:nth-child(5) {
  flex-grow: 4;
}
{% endblock %}


===

### `flex-shrink`

`flex-shrink` 属性决定这个 item 收缩的比例，默认值是1。

```css
.item {
  flex-shrink: <number>; /* default 1 */
}
```

{% endblock %}
{% block flex-shrink %}
.item {
  min-width: 100px;
  flex-grow: 1;
}

.item:nth-child(3) {
  flex-shrink: 2;
}

.item:nth-child(5) {
  flex-shrink: 1;
}
{% endblock %}

===

### `flex-basis`

`flex-basis` 表示**在分配空闲空间之前** item 的 main size 的初始大小。

它可以使用与 `width`/`height` 属性相同的值，比如`20px`, `30%`, `4em` 等。除此之外，还有一个值 `content`， 表示尺寸基于 item 的内容, 相当与`width: auto; flex-basis: auto;`。

默认值是 `auto`，表示 如果该 item 定义了 main size 尺寸则使用该尺寸，如果没有，则基于其内容。

{% endblock %}

{% block flex-basis %}
.item:nth-child(2) {
  width: 20px;
  flex-basis: auto;
}

.item:nth-child(4) {
  flex-basis: 20%;
}
{% endblock %}

===

{% block markdown %}

### `flex`

`flex` 是 `flex-grow`, `flex-shrink`, `flex-basis` 三个属性的简写形式。

常见值：

`initial`

等价于 `0 1 auto`。 item 的大小会遵守其 `width`/`height` 属性，如果 mian size 计算出来的是 `auto`，则 item 的大小是由其内容决定的。如果 container 仍然有可用空间，item 并不会伸展。但如果 container 的空间不足， item 会尽可能的缩小。

`auto`

等价于 `1 1 auto`。item 的大小会遵守其 `width`/`height` 属性，但该 item 可以伸展。假设 container 中所有的 item 使用都是 `flex: auto`, `flex: initial` 或者 `flex: none`，当所有 item 的尺寸计算完毕以后，空余的空间会被平均分配给属性 `flex: auto` 的 item。

`none`

等价与 `0 0 auto`。item 的大小会遵守其 `width`/`height` 属性，但不会有伸缩性。

`flex: <positive-number>`

等价与 `flex: <positive-number> 1 0`。该 item 会完全按照指定的比例去使用空闲空间。

{% endblock %}

### `align-self`

`align-self` 属性可以用来设置该 item 在 cross axis 上的对齐方式，用来覆盖在 flex container 上的相关属性设置。

其可用值与 *container* 上 `align-items` 的值一致。

```css
.item {
  align-self: auto | flex-start | flex-end | center | baseline | stretch;
}
```

{% endblock %}

{% block align-self %}
.item:nth-child(2) {
  align-self: flex-end;
}

.item:nth-child(3) {
  align-self: center;
}

.item:nth-child(7) {
  font-size: 10px;
  align-self: baseline;
}

.item:nth-child(8) {
  font-size: 25px;
  align-self: baseline;
}
{% endblock %}
